Инструкция по запуску
------
Примечание: должен быть установлен npm
1. Скачайте архив
2. В консоле перейдите в папку "/modelofsawingworkshop"
3. Пропишите:
```js
npm run serve
```
4. Программа запущена. В браузере переходим на http://localhost:8080
***
